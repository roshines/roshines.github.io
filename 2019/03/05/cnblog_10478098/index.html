判断闰年

2019-03-05
笔记

这是👉判断闰年👈的摘要

c语言代码实现如下：

#include <stdio.h>
int main() {
int year;
scanf(“%d”, &year);
if((year % 100 !=0 && year % 4 == 0) || year % 400 ==0){
printf(“YES”);
}else{
printf(“NO”);
}
return 0;
}
